MovieBase

This API provides a data base of movies including their title, genre, director, release date, description, main cast and a poster/cover.

These are the endpoints of the API:

Request URL Response
GET list of all movies /movies A JSON holding all the titles as objects, e.g.:
{title: Terminator,
director: James Cameron
genre: Action}
GET list of all users /users A JSON holding all the users as objects, e.g.:
{title: User 1,
id: 1}
GET list of all directors /directors A JSON holding all the directors as objects, e.g.:
{name: James Cameron,
birthDate: 1954}
GET user by name /users/:username A JSON objects with user' data, e.g.:
{name: Wojtek Lukowski,
id: 1}
GET movie by title /movies/:title A JSON object with title's data, e.g.:
{title: Terminator,
director: James Cameron
genre: Action}
GET director by name /directors/:name A JSON object with directors's data, e.g.:
{name: James Cameron,
birthDate: 1954}
GET genre by title /movies/:title A JSON object with directors's data, e.g.:
{genre: Action}
